# MERN Stack Task Manager Assessment

This is a full-stack Task Management application built to demonstrate proficiency with the MERN (MongoDB, Express, React, Node) stack. The application allows users to add, view, edit, and delete tasks, with real-time updates and persistence handled by the backend.

## ‚ú® Key Features

* **Task Management (CRUD):** Complete Create, Read, Update, and Delete functionality for tasks.
* **Time Tracking Logic:** Implementation of logic to calculate and display the time elapsed since a task was created (e.g., "Just now," "2 hours ago").
* **Responsive UI:** Built with modern React components for a clean and responsive user experience.
* **Prisma ORM:** Utilizes Prisma to manage and interact with the PostgreSQL database.
